在有anaconda环境的电脑上打开方法：
1.解压压缩包到任意位置
2.在文件资源管理器里找到文件夹所在路径并复制
3.以管理员身份运行Anaconda Prompt
输入以下指令：
pip install pygame
等待安装好
4.命令行中继续输入
cd 刚才复制的路径
5.命令行中输入
python alien_invasion.py
即可开始游戏！

游戏帮助：
1.鼠标点击屏幕中心的Play按钮即可开始游戏
2.按键盘上的左右键控制飞船移动，空格键发射子弹
3.按Q键退出游戏
